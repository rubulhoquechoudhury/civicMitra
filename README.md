# CivicMitra

## About

CivicMitra is a civic issue reporting platform. Citizens can submit reports (potholes, garbage, street lights, etc.) with optional photo evidence. Authorities get a dashboard to view and manage reports by state and district.

## Why We Built It

Most civic complaints go unheard because there’s no simple, structured channel to report them. Existing systems are either informal (social media, word of mouth) or overly complex. We built CivicPulse to provide a direct, easy-to-use link between citizens and local authorities.

## The Need

- **Citizens** need a quick way to report local issues without bureaucracy.
- **Anonymous reporting** encourages people to report without fear.
- **Authorities** need a single place to see, prioritize, and act on reports.
- **Location-based data** (state, district) helps route issues to the right departments.

## Features

- Citizen report submission with photo upload
- Optional anonymous reporting
- Authority dashboard with report management
- State and district-based filtering
